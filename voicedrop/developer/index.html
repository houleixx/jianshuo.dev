三个服务

服务Base URL用途
Files APIjianshuo.dev/files/api/账户、录音 / 文章文件、照片、分享、公众号、社区。绝大多数调用在这里。
Agent Workerjianshuo.dev/agent/触发挖矿、实时状态推送（WS）、语音改稿、算力余额、6+4 设备配对。
Reco Workerjianshuo.dev/reco/社区 feed 排序 + 互动上报。可随时拔掉，挂了不影响主流程。

数据流一句话：上传 .m4a → 服务端挖矿（ASR + Claude）写出 articles/<stem>.json → 客户端读文章、分享、发公众号 / 发社区。
